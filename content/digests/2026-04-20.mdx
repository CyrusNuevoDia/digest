**TL;DR.** YC founder Sherwood Callaway is building Sazabi, an AI-native observability platform that ditches dashboards and monitors entirely in favor of autonomous agents that detect, root-cause, and fix production issues.

**Takeaways**

- Pitch is "Cursor for production" / "self-driving car of observability" — no dashboards, no monitors, no manual log search; the agent does everything via plain-language chat.
- Founder thesis: AI coding agents are shipping 3-5x more code (and bugs), but monitoring tools like Datadog and Sentry haven't evolved — too much setup, noisy alerts, slow root-causing.
- Claims Sazabi beat Claude Code on Terminal Bench 2; currently in the YC batch, came out of stealth late Feb, ramping beta onboardings this week.
- Can integrate with coding agents like Cursor to auto-implement fixes, aiming toward "self-healing software."
