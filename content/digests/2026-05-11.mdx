**TL;DR.** A "Rosetta stone" for RL environments — the same three envs (Jupyter agent, Wordle, desktop computer-use) reimplemented across six frameworks so you can see how each one actually models tools, state, and rollouts.

**Takeaways**

- Covers OpenEnv, ORS, NeMo Gym, Verifiers, SkyRL Gym, and GEM — each folder is a runnable minimal example, not a training run, so you can compare dialects side by side.
- Ships 5 agent skills (`.claude/skills/`) that turn a plain-English env description into runnable code across frameworks; installable via `npx skills add adithya-s-k/RL_Envs_101` and works with Claude Code, Cursor, Codex, etc.
- Companion to a Hugging Face blog post; 82 stars, Python, MIT-ish hands-on guide vibe — useful if you're deciding which RL env framework to commit to before writing real training code.
