**TL;DR.** Build a "code factory" where agents write code and a review bot gates merges — but only if you enforce current-head SHA discipline, deterministic preflight gates, and machine-verifiable evidence.

**Takeaways**

- Current-head SHA matching is non-negotiable: stale "clean" review evidence will merge broken PRs if you don't pin checks to the latest commit.
- Run a cheap `risk-policy-gate` before the expensive CI fanout — don't burn minutes on PRs already blocked by policy.
- Use exactly one canonical rerun-writer workflow with SHA dedupe to kill duplicate bot comments and race conditions.
- For UI changes, require evidence manifests (flows, entrypoints, identity, freshness) in CI — screenshots in PR text don't count.
