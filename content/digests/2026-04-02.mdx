**TL;DR.** Karpathy uses LLMs to compile a personal markdown wiki in Obsidian from raw sources, then queries and grows it agentically — shifting his token spend from code to knowledge.

**Takeaways**

- Pipeline: drop sources into `raw/`, let an LLM "compile" them into linked .md articles with summaries, backlinks, and categories; Obsidian is the IDE, the LLM does all the writing.
- At ~100 articles / 400K words, fancy RAG wasn't needed — auto-maintained index files and summaries let the agent find what it needs; outputs come back as markdown, Marp slides, or matplotlib, then get filed back into the wiki so explorations compound.
- "LLM health checks" lint the wiki for inconsistencies, missing data, and new article candidates; he sees room for a real product here, and eventually finetuning so the model "knows" the wiki in its weights.

**TL;DR.** Ryan Carson's "clawchief" is an operating layer on top of OpenClaw that turns it into a proactive chief of staff handling email triage, scheduling, CRM updates, and daily task prep.

**Takeaways**

- Repo at github.com/snarktank/clawchief — skills + workspace files layered on an existing OpenClaw + GOG (Gmail/Calendar/Sheets) install, not a replacement.
- `HEARTBEAT.md` is the proactivity trigger: makes the assistant loop through priority map, auto-resolver, and live task list, and only ping when action is actually needed.
- One canonical `tasks.md` as the single source of truth — kills duplicates and stale state instead of guessing from chat history.
- Credits @pedroh96's priority-map / auto-resolver / ingestion pipeline pattern (from his Core Memory interview) as the conceptual backbone.

**TL;DR.** Warp hit 500K weekly devs by hunting untapped channels (YouTube creators, paid GitHub READMEs, $2 Google Ads CAC, hackathon judges) before anyone else noticed them.

**Takeaways**

- Don't build a content team — hire the creator already making what your audience loves and get out of their way (0→60K YouTube subs in 3 months).
- Paid GitHub README placements worked so well people thought GitHub had launched an ad network.
- Cracked Google Ads down from $30 to $2 CAC by finding one sleeper keyword ("best Mac terminal") and relentlessly A/B testing landing pages.
- At hackathons, market to the judges, not the students — they're the senior leaders at your target accounts.
